## Content Description
This is an up-to-date, bilingual educational guide about the Lightning Network - Bitcoin's Layer 2 scaling solution. The content covers:

- Introduction to Lightning Network fundamentals
- Payment channels: opening, updating, and closing
- Channel capacity and liquidity management
- HTLC (Hash Time-Locked Contracts) technical details
- BOLT standards (Basis of Lightning Technology)
- Channel closure mechanisms and security

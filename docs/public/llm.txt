## What This Site Is About

Lightning Network Guide (ln.guide) is a comprehensive, free educational resource that explains how the Lightning Network works. Lightning Network is Bitcoin's Layer 2 scaling solution that enables fast, low-cost payments through payment channels.

## Key Topics Covered

1. **Lightning Network Basics**
   - What is Lightning Network
   - Layer 2 scaling concept
   - Benefits: speed, low fees, micropayments, privacy

2. **Payment Channels**
   - How channels work
   - Opening channels (funding transactions)
   - Updating channel state
   - Closing channels (cooperative and unilateral)

3. **Technical Concepts**
   - HTLC (Hash Time-Locked Contracts)
   - Multisig addresses (2-of-2)
   - Commitment transactions
   - Revocation keys
   - Watchtowers

4. **BOLT Standards**
   - BOLT 1-12 specifications
   - Messaging protocols
   - Channel operations
   - Payment routing
   - Invoice formats

5. **Liquidity Management**
   - Inbound vs outbound capacity
   - Rebalancing strategies
   - Multi-Path Payments (MPP)
